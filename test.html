Speech2TurboWarp Extension

概要

Speech2TurboWarpは、ブラウザの音声認識APIを使用してTurboWarpで音声入力を可能にする拡張機能です。

使い方

  1. TurboWarpエディターを開く: https://turbowarp.org/editor
  2. 「拡張機能」ボタンをクリック（左下のブロックパレットの一番下）
  3. 「カスタム拡張機能」を選択
  4. このファイルのURLを入力するか、ローカルHTTPサーバー経由でアクセス

ローカルHTTPサーバーの起動方法

# Python 3の場合
python3 -m http.server 8000

# Python 2の場合
python -m SimpleHTTPServer 8000

# Node.jsの場合（http-serverをインストール済み）
http-server -p 8000

サーバーを起動したら、TurboWarpのカスタム拡張機能URLに以下を入力:

http://localhost:8000/speech2turbowarp.js

重要な注意事項

  • 音声認識にはマイクへのアクセス許可が必要です
  • HTTPSまたはlocalhostでのみ動作します
  • 対応ブラウザ: Chrome, Edge, Safari（一部）
  • 日本語認識に設定されています（変更可能）

ブロックの説明

  • 音声認識開始: 音声認識を開始します
  • 音声: 認識された音声テキストを返します
  • 音声認識中: 現在音声認識中かどうかを返します
  • 音声認識停止: 音声認識を停止します
  • 音声が認識されたとき: 音声が認識されたときに実行されるイベントブロック

サンプルプロジェクト

以下のようなプログラムを作成できます:

[音声が認識されたとき]
もし <(音声) = [こんにちは]> なら
  [こんにちは！] と言う
でなければ もし <(音声) = [ありがとう]> なら
  [どういたしまして！] と言う
